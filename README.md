# AfroMint - Automated African News Website

This is a fully automated, mobile-first, and lightweight news website designed for Africa.

## Features
- **Auto-Update**: Fetches news every 6 hours.
- **AI Rewritten**: Simple English (Class 2 Level) for easy reading.
- **Fast**: Loads in <1 second, optimized for 2G/3G.
- **Zero Maintenance**: Runs on GitHub Actions.
